
## Usage <a id="usage"></a>

```js
// Import using TypeScript/ES6+
import { serverlessHello } from '@seantrane/serverless-hello';
// or, require using basic JavaScript:
const serverlessHello = require('@seantrane/serverless-hello');

serverlessHello();
// => "Hello world!"
```
